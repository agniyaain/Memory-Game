# Memory Game

## Deskripsi  
Memory Game adalah sebuah permainan kartu “match-pair” sederhana berbasis HTML, CSS, dan JavaScript.  
Tujuan dari proyek ini — selain sebagai latihan implementasi web dasar — adalah untuk membuat game interaktif di mana pemain harus mencocokkan pasangan kartu yang identik.  

## Cara Main  
- Permainan dimulai dengan semua kartu tertutup.  
- Pemain membuka dua kartu setiap giliran.  
- Jika kedua kartu cocok → kartu akan tetap terbuka.  
- Jika tidak cocok → kartu kembali tertutup.  
- Permainan berakhir ketika semua kartu sudah berhasil dicocokkan.  

## Teknologi  
- HTML  
- CSS  
- JavaScript  

## Cara Menjalankan Secara Lokal  
1.    Clone repository ini:
   ```bash
   git clone https://github.com/agniyaain/Memory-Game.git
   ```
2.   Masuk ke folder project:
   ```bash
   cd Memory-Game
   ```
3.   Buka file index.html di web browser favoritmu — modern dan up-to-date browser disarankan.
